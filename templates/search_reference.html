Search Reference



Filters
Filters are search keywords that will filter and exclude posts.

    before:<date>    -  Filter posts before a time and date
    after:<date>     -  Filter posts after a time and date
    user:<username>  -  Filter posts by user


Date Keywords
    today
    week
    month
    year
    yesterday
    yesterweek
    yestermonth
    yesteryear


Examples
    after:today       -  Show all posts posted today
    before:today      -  Show all posts posted before today
    after:10/10/2001  -  Show all posts posted after 10/10/2001
    after:Oct-10-2001 -  Show all posts posted after 10/10/2001
    after:week        -  Show all posts posted this week
    user:john         -  Show all posts posted by the user 'john'





Tags
Tags are an implicit filter. All keywords that do not include the ':' character are treated like tags. Tags are separated by spaces, NOT BY COMMAS. When using the search, only the posts that match all of the tags will be displayed.


Examples
    cats
    dogs
    politics
    etc...





Exclusions
To invert or exclude tags or filters, you can use the exclusion character '-'.


Examples
    -user:john      -  Excludes all posts from the user 'john'
    -cats           -  Excludes all posts with the tag 'cats'





Sorting
Unlike filtering, the sort keyword will not exclude any posts, but will instead change the ordering of posts.

    sort:newest     - sort posts by the newest published post
    sort:oldest     - sort posts by the oldest published post
    sort:rating     - (soon to be added)



Example Searches

    cats cake -dogs -user:jonny59 after:yestermonth sort:oldest
